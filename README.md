# Advent of Code

This is my attempt at <https://adventofcode.com/2021>.

I've decided to try it in Typescript as I'd like to get a little better at it.

# Getting started

Install deps:

```
$ nvm install
$ npm install
```

Run tests:

```
$ npm run test
```

Run one day:

```
$ npm run test:watch -- --grep 'day 17'
```
